## 정교화 원칙

1. **결정 대기 항목은 명시적으로 표시** — 추측으로 채우지 않는다.
2. **모든 자산은 실제 티커로 표기** — 백테스트 착수 가능성 확보.
3. **MVP는 룰 기반·고정비중** — AI/최적화/에이전트는 뼈대 위 교체로 붙인다.
4. **규제 회피** — MVP는 "분석·교육용 리포트"로 한정, 실주문·일임 배제.
