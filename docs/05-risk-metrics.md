# 05. 리스크 지표 정의서

Risk Agent · 백테스트 엔진 · Compliance 가드레일이 공유하는 **단일 진실 원천(single source of truth)**.
지표 정의가 곳곳에서 달라지면 리포트와 차단 로직이 어긋나므로, 계산식·파라미터·기준을 여기서만 정의한다.

## 0. 공통 규약

- **수익률 정의**: 로그수익률 아님, **단순수익률** `r_t = P_t / P_{t-1} - 1` 기본. 연율화 시 명시.
- **통화(D4 연동)**: 리스크는 **현지통화 기준으로 계산**한다. KRW 표시 요청 시, 원화 환산 수익률 시계열에 동일 지표를 **재계산**(환효과 포함)한다. 두 값을 모두 저장.
- **관측 주기**: 일별. 연율화 계수 = 252 (거래일). 월별 계산 시 12.
- **기준 데이터 창**: 지표별로 명시(전체기간 / 롤링 / 최근 N년).
- **재현성**: 동일 입력·동일 코드 버전 → 동일 결과(KPI: 재현성 100%). 난수 사용하는 몬테카를로는 시드 고정.

## 1. 지표 정의

### 1.1 변동성 (Volatility)
- 정의: 일별 수익률 표준편차의 연율화.
- 식: `vol_annual = std(r_daily) * sqrt(252)`
- 창: 전체기간 + 롤링 1년(252일) 병행 산출.

### 1.2 최대낙폭 (MDD, Maximum Drawdown)
- 정의: 누적가치 곡선의 고점 대비 최대 하락률(음수).
- 식:
  ```
  cum_t = ∏(1 + r_i)  (i=0..t)
  peak_t = max(cum_0..cum_t)
  dd_t = cum_t / peak_t - 1
  MDD = min(dd_t)
  ```
- 부가: MDD 발생 구간(peak일~저점일)과 **회복 소요기간(recovery)** 함께 기록.

### 1.3 VaR (Value at Risk)
- 정의: 신뢰수준 α에서 기간 손실의 분위수.
- **기본 파라미터**: α = 95%, 보유기간 = 1일 및 1개월(21일) 두 종.
- 방법(둘 다 산출, 리포트는 Historical 기본):
  - **Historical VaR**: 과거 수익률 분포의 (1-α) 분위수. `VaR_95 = -quantile(r, 0.05)`
  - **Parametric VaR**: 정규가정. `VaR_95 = -(μ + z_0.05 * σ)`, z_0.05 = -1.645
- 표기: 양수 손실률로 표기(예: VaR 95% = 2.1%).

### 1.4 CVaR / Expected Shortfall
- 정의: VaR를 초과하는 손실들의 평균(꼬리위험).
- 식(Historical): `CVaR_95 = -mean( r | r <= quantile(r, 0.05) )`
- VaR와 동일 α·보유기간으로 산출.

### 1.5 하방변동성 & Sortino (참고 지표)
- 하방편차: 목표수익 0(또는 무위험) 이하 수익률만의 표준편차.
- Sortino = (연평균초과수익) / (하방편차 연율화).

### 1.6 위험조정성과
- **Sharpe** = (연평균수익 - 무위험) / 연변동성. 무위험 = 해당 통화 단기금리(USD: 3M T-Bill, KRW: CD금리).
- **Calmar** = 연평균수익 / |MDD|.

### 1.7 민감도 (Risk Agent 시나리오 입력)
- **FX 민감도**: USD/KRW ±10% 충격 시 KRW 기준 포트폴리오 가치 변화(해외자산 비중 × 환노출 비중).
- **금리 민감도**: 채권 자산의 듀레이션 기반 근사 `ΔP ≈ -D * Δy` (D는 ETF 듀레이션, IEF≈7.5, TLT≈17, SHY≈1.9 참고).
- **주식충격**: 주식군 -30% 시 포트폴리오 영향(비중 가중).

### 1.8 집중도
- **자산군 집중**: 최대 단일 자산군 비중.
- **단일 ETF 집중**: 최대 단일 ETF 비중.
- **국가/통화 집중**: 해외자산 통화별 노출 합.

## 2. 스트레스 시나리오 (필수 세트)

과거 실측 구간 + 가상 충격을 병행한다.

| 시나리오 | 유형 | 정의 |
| -- | -- | -- |
| 2008 금융위기 | 실측 | 2007-10 ~ 2009-03 구간 재현 |
| 2020 코로나 | 실측 | 2020-02-19 ~ 2020-03-23 |
| 2022 금리인상 | 실측 | 2022-01 ~ 2022-10 |
| 주식 -30% | 가상 | 전 주식군 즉시 -30% |
| 원화 강세 10% | 가상 | USD/KRW -10% (해외자산 원화수익 하락) |
| 미 금리 +100bp | 가상 | 듀레이션 기반 채권 손실 |
| 인플레 재상승 | 가상 | 장기채·성장주 동반 부담 |

각 시나리오는 **포트폴리오 손실률 + 최대 기여 자산**을 함께 출력.

## 3. 성향별 허용 기준 (Compliance 가드레일)

03 문서의 성향 등급별 하드 상한. **초과 시 차단→강등→재계산**.

| 성향 | 목표 연변동성(가이드) | MDD 허용 상한 | 연간 예상손실(≈VaR95 연율)  상한 |
| -- | -- | -- | -- |
| 안정형 | ≤ 6% | ≥ -10% | -8% |
| 중립형 | ≤ 10% | ≥ -18% | -15% |
| 성장형 | ≤ 15% | ≥ -28% | -22% |
| 공격형 | ≤ 20% | ≥ -40% | -32% |

> "연간 예상손실"은 설문 Q6(최대 손실 감내)과 교차 검증한다. **둘 중 더 보수적인 값**을 상한으로 채택.

### 3.1 집중도 상한 (성향 무관 공통)
- 단일 ETF ≤ 30%
- 자산군 최대 ≤ 성향별 주식밴드 상한
- 개별종목(v2) ≤ 5% — 적용 후보 유니버스: [S&P500 5개년 합집합](../tools/sp500/sp500_by_sector.html)(분류 참조용, 가격·수익률 없음)
- 현금 최소 ≥ 3% (유동성 필요 "높음"이면 ≥ 10%)

```json
{
  "calc_currency": "USD",
  "display_currency": "KRW",
  "vol_annual": 0.112,
  "mdd": -0.173,
  "mdd_recovery_days": 210,
  "var95_1d": 0.018,
  "cvar95_1d": 0.026,
  "var95_annual": 0.141,
  "sharpe": 0.68,
  "calmar": 0.41,
  "currency_exposure": {"USD": 0.92, "non_USD": 0.08},
  "concentration": {"max_asset_class": 0.30, "max_etf": 0.30},
  "stress": [
    {"scenario": "2008", "loss": -0.31, "top_contributor": "SPY"},
    {"scenario": "2022", "loss": -0.19, "top_contributor": "TLT"}
  ],
  "breaches": []

```

> **스키마 정정(2026-07-03 R2)**:
> 1. **`var95_annual` 필드 신설** — §3 성향별 상한("연간 예상손실 ≈VaR95 연율")과 03 §4 Q6 교차검증이 **이 값을 차단 기준으로 쓰는데 기존 스키마엔 `var95_1d`만 있어 compliance를 못 먹였다.** 계산은 몬테카를로가 아니라 **Historical VaR95를 √기간으로 스케일하거나 실측 스트레스 MDD를 대용**(경량). §5 체크리스트에 `var95_annual` 산출·상한대조 항목 추가.
> 2. **통화 필드 분리** — 단일 `currency`를 `calc_currency`(계산 기준, 현지통화=USD, §0)·`display_currency`(표시)로 분리. **MVP는 USD 계산 + KRW 표시 환산 1세트**(현지·KRW 지표 이중 전량 재계산·저장은 08 §6 범위 조정으로 v2). compliance 판정은 항상 `calc_currency` 기준.
> 3. **`rate_sensitivity`(금리 듀레이션)·`fx_sensitivity`(±10% 충격) 제거 → `currency_exposure`(통화노출 비중)로 대체** — MVP 축소(08 §6). 금리·주식충격 정량 민감도는 §1.7 정의는 유지하되 **v2**. Q9 환회피 정합성은 `currency_exposure`% 정성 고지로 충족.

## 5. 검증 체크리스트

- [ ] VaR/CVaR가 같은 α·보유기간에서 CVaR ≥ VaR 성립하는지
- [ ] MDD가 [-1, 0] 범위 내인지
- [ ] **`var95_annual`이 산출되고 §3 성향별 상한과 대조되는지 (compliance 차단 기준)**
- [ ] (MVP) `calc_currency`(USD) 기준 지표 + `display_currency`(KRW) 표시 환산이 일관되는지 — 이중 전량 재계산·저장은 v2
- [ ] 스트레스 손실이 정상 시장 변동성보다 큰지(방향성 sanity)
- [ ] 성향별 상한 위반 시 breaches 배열에 사유·초과폭 기록되는지
