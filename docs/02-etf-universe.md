# 02. ETF 유니버스 1차 목록

원칙: 20년 이상 데이터 확보 가능 + 유동성 충분 + 자산군 대표성. 상위 기획안 자산군 분류를 **실제 티커**로 확정한다.
(MVP 데이터 소스가 미국 상장 ETF 기준이면 미국 티커, 국내 상장 기준이면 KRX 티커로 매핑 — D7 결정에 따름.)

## 0. 자산군 분류 체계 (요약)

아래 4단 계층이 이 문서 티커 목록의 뼈대다. **각 자산군의 역할·리스크 팩터·성향별 밴드 등 분석 계층은 [07-asset-classes.md](07-asset-classes.md)를 정본으로 한다.**

| 대분류 | 중분류 | 자산군 슬롯 | 대표 티커(미국) |
| -- | -- | -- | -- |
| 주식 | 미국 광의 | 미국 대형주 | SPY |
| 주식 | 미국 광의 | 미국 성장주(기술) | QQQ |
| 주식 | 해외 선진 | 선진국(미국 외) | EFA |
| 주식 | 해외 신흥 | 신흥국 | EEM |
| 주식 | 스타일 | 배당주 | VYM / SCHD |
| 채권 | 국채(만기) | 중기국채 | IEF |
| 채권 | 국채(만기) | 장기국채 | TLT |

| 현금성 | 단기 | 단기채/현금성 | SHY |
| 실물 | 귀금속 | 금 | GLD |

> MVP 고정비중 슬롯(§3)은 이 체계의 부분집합(SPY·QQQ·EFA·EEM·IEF·TLT·SHY·GLD)이다. 배당주·종합채권은 유니버스에는 두되 MVP 배분에서는 제외.

> **20년+ 이력은 각 슬롯의 대표(첫) 티커 기준**. 대안 티커 중 VOO(2010)·VEA(2007)·BND(2007)·BIL(2007)·SCHD(2011)는 20년 미만 → 상장 이전 구간은 §5 splice 정책으로 대체하거나 백테스트는 대표 티커로 수행한다. MVP 고정비중 8슬롯 대표 티커(SPY·QQQ·EFA·EEM·IEF·TLT·SHY·GLD)는 모두 20년 확보.

## 4. MVP 제외 (2단계 이후)

- 섹터 ETF, 팩터 ETF(모멘텀/퀄리티/저변동), 하이일드, 리츠, 원자재 광범위
- 개별 종목 (S&P500 대형주, KOSPI200 대형주) — v2 위성 슬리브 후보. S&P500 5개년 합집합 593종의 테마 분류는 [tools/sp500 유니버스](../tools/sp500/sp500_by_sector.html)에 구현(정본 07 §6)
