# 04. IPS(투자정책서) 템플릿

IPS Agent의 출력 규격. Investor Agent 출력(03)을 받아 아래 필드를 채우고 문장 템플릿으로 렌더링한다.
CFA Institute 권고에 따라 계획·실행·모니터링·시장혼란기 행동 기준을 포함한다.

## 1. IPS 필드 (구조화)

| 필드 | 소스 | 예시 |
| -- | -- | -- |
| 투자 목적 | Q3 | 장기 자산증식 |
| 투자 기간 | Q2 | 8년 이상 |
| 목표 수익률(참고) | 성향별 기준 | 연 5~7% |
| 허용 손실 범위 | Q6 | 연 -20% |
| 자산군 허용 범위 | 성향별 밴드 | 주식 51~75% |
| 기준 배분(SAA) | 모델포트폴리오 | 주식75/채권15/금5/현금5 |
| 리밸런싱 기준 | 고정 규칙 | 목표 대비 ±5%p 이탈 시 검토 |
| 환헤지 정책 | Q9 | **MVP 환헤지 미지원** (헤지형 슬롯 v2) |
| 통화 노출(%) | 배분 | USD 약 92% / 원화 8% (원화강세 시 손실 확대) |
| 응답 반영 고지 | Q6·Q8·Q9 | "환회피 응답 반영: 원화 벤치 비교 강조·헤지형 v2 로드맵" |
| 스트레스 공시 맥락 | 백테스트 | "2008/2020/2022는 극단 참고 공시(평시 상한 아님)" |
| 금지 자산 | 컴플라이언스 | 레버리지·인버스 ETF |
| 성과 평가 기준 | 벤치마크 | 60/40 대비 위험조정성과 |
| 변경 조건 | 규칙 | 성향/목적/기간 변동 시 재작성 |
| 시장 혼란기 행동 | 규칙+Q10 | 밴드 이탈 전 임의 매도 금지 + **하락기 행동 대본**(Q10 반영) |

## 2. 문장 템플릿 (렌더링 예시)

```text
본 문서는 {profile} **유형**에 대응하는 예시 배분이며 개인별 투자권유·자문이 아니다(교육·분석용).
해당 유형은 {horizon}년 이상의 {objective}을 전제로 {saa} 기준 배분을 사용한다.

목표 비중에서 ±{band}%p 이상 이탈할 경우 리밸런싱 검토 대상이 되며,
평시 기준 예상 연손실이 {max_loss}(성향 상한과 Q6 감내 중 보수값)를 초과하는 조합은 예시에서 제외한다.

[환·통화] 본 배분은 해외자산 비중이 높아 **USD 노출 약 {usd_exposure}%**이며 원화강세 시 손실이 확대될 수 있다.
**MVP는 환헤지를 제공하지 않는다**(헤지형 슬롯 v2). 귀하의 환노출 선호({fx_pref}) 응답은 {fx_reflection}로 반영했다.

[응답 반영] 귀하의 응답 중 {reflected_answers}가 위 배분·고지에 반영되었습니다.

[스트레스 안심 프레이밍] 아래 2008/2020/2022 손실은 20년에 몇 번 오는 극단 시장의 **참고 공시**이며(평시 상한과 다름),
{profile} 유형도 이 구간엔 크게 하락할 수 있으나 역사적으로 회복해 왔습니다(미래 수익 보장 아님).
```

> **R4(2026-07-05) — 응답→산출 연결·기대관리 필수**: (1) `{reflected_answers}`·`{fx_reflection}`처럼 **투자자 응답이 산출에 반영된 흔적**을 렌더한다("물어보고 안 지킴" 배신감 방지, 행동재무 R4). (2) 환효과 분리(자산손익 vs 환손익)를 **환회피 응답자에겐 기본 표기**(선택 아님, 화폐환상 통제). (3) disclosed 스트레스 숫자 옆에 **안심 내러티브**를 라벨만큼의 비중으로(공포 숫자 단독 노출 금지). (4) IPS·SAA·MDD·VaR 등 용어는 **툴팁/용어집(progressive disclosure)** 동반.

## 3. 버전·감사

- 모든 IPS는 `ips_documents` 테이블에 버전 저장 (상위 기획안 4.3)
- 재작성 트리거(성향/목적/기간 변경) 발생 시 신규 버전 생성, 이전 버전 보존
- 감사 로그에 생성 시점 입력 스냅샷·모델버전 기록
