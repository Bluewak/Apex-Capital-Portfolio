# 01. 요구사항 정의서 & 핵심 의사결정

## 1. 서비스 한 줄 정의

투자자 설문을 받아 **투자정책서(IPS) → 자산배분 → 리스크 리포트 → 설명 리포트**를 자동 생성하는 분석형 포트폴리오 리포트 서비스. (MVP 단계는 실주문·일임 없음)

## 2. MVP 범위 (In / Out)

### In scope (1차년도 MVP)
- 투자자 설문 및 위험성향 분류 (룰 기반)
- IPS 자동 생성 (템플릿 기반)
- 성향별 고정 모델포트폴리오 4종 (안정/중립/성장/공격)
- ETF 유니버스 기반 자산배분
- 20년 이상 백테스트 (2008 금융위기·2020 코로나·2022 금리인상기 포함)
- 기본 리스크 리포트 (변동성/MDD/VaR/CVaR)
- 설명 리포트 (룰 기반 문장 → 이후 LLM 교체)

> **백테스트 기간 20년 하한 근거**: 15년(≈2011~)은 2008 금융위기를 놓친다. 스트레스 정의(05 §2)의 2007-10 고점부터 재현하려면 최소 ~19년 → **20년을 하한**으로 고정(≈2006~현재). 코어 8개 슬롯 대표 티커(SPY·QQQ·EFA·EEM·IEF·TLT·SHY·GLD)는 모두 2004년 이전 상장이라 실데이터로 20년 확보 가능(02 §1).

### Out of scope (MVP 제외 — 상위 기획안 12장 준수)
- 개별 종목 추천 / 딥러닝 가격 예측 / 뉴스 감성
- 실주문·자동매매·일임운용
- 세금 최적화 / 금융사 계좌 연동
- 멀티에이전트 토론 / Black-Litterman / CVaR 최적화 (2차년도 이후)

### 4-1. 통화 토글 규격 (D4 상세)

- **계산은 항상 현지통화(USD 등) 기준**으로 수익률·리스크 산출 → 왜곡·이중환산 방지.
- **표시 단계에서 통화 선택**: 사용자가 `KRW` / `USD` 토글. 기본값 = KRW.
  - KRW 표시 = 현지통화 수익률에 해당 구간 USD/KRW 환율 적용 (환효과 포함).
  - USD 표시 = 현지통화 수익률 그대로.
- 리포트·대시보드·백테스트 그래프 모두 동일 토글을 따른다.
- 환효과가 큰 구간은 리포트에서 "환율 기여분"을 분리 표기(선택).

> D2가 "미국 우선"이므로 초기 유니버스는 미국 상장 ETF 중심(02 문서 1절)으로 확정, 국내 상장 매핑(02 문서 2절)은 실제 추천 상품 연결 시 사용.

## 4. 사용자 흐름 (MVP)

```text
설문 입력 → 위험성향 분류 → IPS 생성 → 모델포트폴리오 배정
→ 백테스트 → 리스크 계산 → 설명 리포트 → PDF/웹 출력
```

## 5. MVP 수용 기준 (Acceptance Criteria)

- 설문 완료 후 포트폴리오+리포트 생성 3분 이내
- 백테스트: 2008 금융위기 / 2020 코로나 / 2022 금리인상 구간 개별 성과 표시
- 벤치마크(S&P500, 60/40, KOSPI200) 대비 위험조정성과 비교
- 모든 추천에 자연어 설명 1회 이상 첨부 (100%)
- 투자자성향 위반 포트폴리오 배정 0건

## 6. 비범위 안전장치 (표현 금지 목록)

- 수익률 보장 표현 / 백테스트 수익률 과장
- "반드시 오른다" 류의 예측 단정
- 레버리지·인버스 ETF 장기 기본 추천
